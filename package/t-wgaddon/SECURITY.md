# Security Policy / Biztonsagi iranyelvek

## Magyar

Ez egy nem hivatalos, fuggetlen TavIR kiegeszito Webgalamb 8 rendszerhez.

Biztonsagi hibajelenteshez:

- E-mail: info@tavir.hu
- Web: https://www.tavir.hu/
- Repository: https://github.com/tavir/webgalamb-tavir-addon

Eles hasznalat elott erossen javasolt:

- egyedi `WGADDON_PASSWORD` beallitasa a `files/wg8conf.php` fajlban,
- debug es PHP kepernyos hibakiiras kikapcsolasa,
- teljes fajl- es adatbazismentes,
- visszaallitas tesztelese masolati kornyezetben.

## English

This is an unofficial, independent TavIR addon for Webgalamb 8.

To report a security issue:

- E-mail: info@tavir.hu
- Website: https://www.tavir.hu/
- Repository: https://github.com/tavir/webgalamb-tavir-addon

Before production use, it is strongly recommended to:

- set a unique `WGADDON_PASSWORD` in `files/wg8conf.php`,
- disable debug and PHP display errors,
- create full file and database backups,
- test restore in a copied environment.
